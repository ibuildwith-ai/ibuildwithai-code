Join me at GitHub Copilot Dev Days | Los Angeles for my session, Agent Skills Foundations and Best Practices.

In this session, I’ll introduce what agent skills are, how they help extend AI agents with repeatable workflows and domain-specific capabilities, and why they matter as developers and builders move beyond simple prompting into more structured AI-assisted development. We’ll look at how agent skills can make tools like GitHub Copilot more useful, consistent, and aligned with real-world work.

### Key Topics Covered

- What agent skills are and why they matter
- How agent skills support repeatable AI-assisted workflows
- Best practices for designing useful, focused, and reusable agent skills
